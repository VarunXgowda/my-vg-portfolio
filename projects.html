Featured Project

Kaizen Geo-Metallurgical Framework (K.G.F.)

AI & Industrial Automation 2024

A predictive soft sensor system that transforms iron ore flotation control from reactive to proactive using machine learning. The framework enables continuous process optimization by predicting silica content with high accuracy 30-60 minutes in advance.

Key Achievements

  • Developed CatBoost Regressor model with R² score of 0.7399 and RMSE of 0.4969%
  • Enabled 30-60 minute predictive lead time for process optimization
  • Identified critical features like Starch-to-Pulp ratio as key drivers
  • Built resilient system with minimal performance degradation on volatile data
